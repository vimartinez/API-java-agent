# API-java-agent
Proyecto de ejemplo para desarrollar APIs en java en agent mode con Cusor.
https://cursor.com/home

## Seguir los siguientes pasos:
- Clonar el repo y abrirlo en el editor deseado
- Editar los siguientes archivos:
  - **requirements.md** especificar los requerimientos deseados en modo historia de usuario breve.
  - **architecture.md** agregar / modificar las pautas de arquitectura de la aplicación.
  -  **api-contract.md** establecer el contrato de la API.
-  El archivo **AGENTS.md** contiene los lineamientos principales
-  Colocar en modo agente y solicitar la generación del proyecto base.
-  Iterar con el agente hasta ir generando todos los componentes.
